# Linux-setup-script to automate the installation of multiple programs

## Programs </> : 

### Dev script 👨‍💻: 
- Chromium 🌐
- Vscode (and extension)
- Python3 🐍
- NodeJS 🟢

### CTF script 🐱‍💻:
- Chromium 🌐
- Ghidra 🐉
- Wordlist 📙
- Wireshark 🦈

This program require an internet connection
It can be launched with ```./setup.sh``` in terminal
